# Enterprise Cyber Security Governance Framework for a Financial Institution

## Project Overview

This project presents an Enterprise Cyber Security Governance Framework designed for a financial institution. It focuses on identifying critical information assets, assessing cybersecurity risks, developing security policies, and recommending governance controls to improve the organization's cybersecurity posture.

## Project Objectives

- Identify critical information assets.
- Perform cybersecurity risk assessment.
- Develop information security policies.
- Improve cybersecurity governance.
- Recommend security controls for financial institutions.

## Project Scope

This project covers cybersecurity governance, asset identification, risk assessment, information security policies, and security recommendations based on industry best practices.
